# GrowEasy AI CSV Importer

An AI-powered CSV Importer built for the GrowEasy Software Developer Internship Assignment.

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js
- Google Gemini API

## Features

- AI-powered CSV field mapping
- Drag & Drop CSV Upload
- CSV Preview
- AI Processing
- CRM Data Table
- Responsive UI
- Error Handling

## Installation

### Frontend

```bash

npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
